Public API

EvidentialDeepLearning.Dirichlet — Type
Dirichlet{N,T}

Statically sized Dirichlet distributions. In contrast to say Distributions.Dirichlet the the number of categories is encoded as a type parameter.

source
EvidentialDeepLearning.DirichletClassification — Type
DirichletClassification(Val(k))

Layer that takes a tensor of floats of size (k*n, b) and turns it into a tensor of Dirichlet distributions of size (n,b).

source
EvidentialDeepLearning.NIG — Type
NIG(μ, ν, α, β)

Normal-inverse-gamma distribution.

source
EvidentialDeepLearning.NIGRegression — Type

NormalInverseGamma regression layer. Takes a tensor of floats of size (4c, n) and outputs a tensor of NormalInverseGamma distributions of size (c,n).

source
EvidentialDeepLearning.posterior_predictive — Method
posterior_predictive(nig::NIG)

Return the posterior_predictive distribution, where the statistical model is the two parameter family Normal(μ,σ²) and the posterior is given by nig.

source

EvidentialDeepLearning.kl_uniform — Method
kl_uniform(d::Dirichlet)

Given P = Dirichlet(α) compute the Kullback Leibler divergence with respect to the uniform Dirichlet distribution:

`D_KL(P || Dirichlet(1,1,1...))`
